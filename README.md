# 跨境Listing优化大师

输入中文产品描述，AI自动生成专业英文Listing。

## 功能特点
- 输入中文，AI自动生成英文标题、卖点、关键词、描述
- 支持 Amazon / Shopee / Temu / eBay 等跨境平台
- 一键复制、导出 TXT/CSV
- 历史记录自动保存
- 多种语气风格可选

## 使用流程
1. 前往 [platform.deepseek.com](https://platform.deepseek.com) 注册并获取 API Key
2. 下载 listing-optimizer.html，双击用浏览器打开
3. 右上角设置填入 API Key
4. 输入中文描述，点击生成

## 文件说明
| 文件 | 说明 |
|------|------|
| listing-optimizer.html | 核心工具 |
| product-page.html | 产品销售展示页 |
